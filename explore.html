How to Use Your Computer

Master the basics of your PC or Mac — file management, navigation, and essential shortcuts.

🖥️ Turning On & Logging In

  • Press the power button to start your computer.
  • Enter your password or PIN at the login screen.
  • Wait for the desktop to fully load before clicking anything.

📁 Managing Files & Folders

  • Open File Explorer (Windows) or Finder (Mac).
  • Create folders to organise your documents and downloads.
  • Right-click to copy, paste, rename, or delete files.

⌨️ Essential Shortcuts

  • Ctrl+C Copy  |  Ctrl+V Paste  |  Ctrl+Z Undo
  • Alt+F4 Close app  |  Win key Open search

🛡️ Staying Safe

  • Install updates when prompted.
  • Never install software from unknown websites.

How to Use the Internet

Navigate the web safely, search effectively, and manage your data usage.

🔍 Searching Effectively

  • Type clear questions into Google or Bing.
  • Use quotes for exact phrases: "affordable internet Zimbabwe".

📧 Using Email

  • Create a free account at Gmail, Outlook, or Yahoo.
  • Attach files using the paperclip icon before sending.

🔒 Staying Safe Online

  • Never share your passwords or ID online.
  • Look for the padlock 🔒 before entering bank details.
  • Avoid clicking links in suspicious emails.

📶 Managing Data Usage

  • Use Wi-Fi whenever possible.
  • Disable auto-play videos on social media apps.

How to Use Your Phone

Get the most from your smartphone — apps, settings, and mobile productivity tips.

📲 Setting Up

  • Insert your SIM and sign in with a Google or Apple account.
  • Connect to Wi-Fi during setup to save data.

📱 Installing Apps

  • Use Google Play Store (Android) or App Store (iPhone).
  • Only install apps from trusted developers.

🔋 Extending Battery Life

  • Lower screen brightness and turn off GPS when not needed.
  • Enable battery saver mode below 20%.

💾 Backing Up

  • Enable Google Backup or iCloud for automatic backups.
  • Google Photos gives you 15 GB free storage.

Learn Microsoft Word

Create professional documents with formatting, templates, tables, and collaboration tools.

📄 Creating a Document

  • Open Word, choose a blank document or template.
  • Use Ctrl+S often. Save as PDF via File → Save As → PDF.

✏️ Formatting Text

  • Highlight text and use the toolbar for Bold, Italic, or Underline.
  • Use Heading Styles for consistent document structure.

📊 Tables & Images

  • Insert → Table to add a grid.
  • Insert → Pictures to embed images.

🖨️ Printing

  • Press Ctrl+P to open print preview, then select your printer.

Learn Adobe

Unlock your creative potential with Photoshop, Illustrator, and more Adobe tools.

🖼️ Photoshop

  • Open an image, use the Crop tool to resize, and work with Layers.
  • Export via File → Export → Export As.

✏️ Illustrator

  • Use the Pen tool for vector shapes; Artboards for multi-page files.
  • Always create logos in Illustrator for infinite scalability.

📄 Acrobat

  • Open, highlight, comment on, and fill PDF forms.

💡 Free Options

  • Adobe Express is free and browser-based for quick designs.
